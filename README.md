# Checkpoint
Checkpointing es una técnica de tolerancia a fallos.

# Descripción
El programa simula un proceso iterativo que guarda su progreso cada 5 iteraciones, automáticamente. Si la ejecución 
se interrumpe, el programa es capaz de reanudarse desde la última iteración o punto de control guardado, evitando la
pérdida de datos.

# Requisitos
- Python 3.8 en adelante
- Libreria pickle y os
